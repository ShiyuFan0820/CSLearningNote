# List

```py
name_list = ['python', 'daily', 'practice', 820, True, [1, 2, 3] ] #using square brackets "[]" at the both ends and using commas "," to separate elements, the elements in the list can be any type, and a new list can be nested in the original list.
print(name_list)
print(type(name_list)) #the type of the list variables are list
```

Run the code:

```py
['python', 'daily', 'practice', 820, True, [1, 2, 3]]
<class 'list'>
```
