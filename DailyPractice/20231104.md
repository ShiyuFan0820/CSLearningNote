# If Statement
```py
x = 7
y = 17
z = x + y
if z > 10:
    print("The number is :", z)
else:
    print("The number is less than 10", z)
```
# For loop
```py
for i in range(5):
    print("Range 5", i)
    
for i in range(5, 15):
    print("Range 5 to 15", i)
    
for i in range(5, 15, 2):
    print("Range 5 to 15 with step = 2", i)
    
for i in range(5, 0, -2):
    print("Range 5 to 0 with step = -2", i)
```
